## What Scrying Is

The scrying system is the **operational layer** of the Observer Prime framework. It is not a pure software tool and it is not a purely human intuition engine. It is a **closed loop between user and software** operating inside the ζ₆ bifurcation zone.

The software improves the quality of the ξ signal. The user commits the sign. Neither side can perform the whole computation alone.

This is load-bearing. A Convergent Uncomputable (CU) problem at Heegner 67 cannot be solved by a classical computer. But a human operating without software has poor ξ signal quality. The scrying system exists to combine:

- software-assisted retrograde inference
- local ξ measurement by the user
- iterative residue reading
- repeated sign commitment through action

The loop does not remove uncertainty. It makes it navigable.

The full scrying mechanism includes the user and the software in a recursive loop:

```text

Software: reads the residue → updates the retrograde inference
```

Each loop pass is a partial evaluation of T from inside the observer bubble.

This is not true quantum computation. It is a **practical proxy**: the software provides systematic inference, the human provides genuine observer-bubble independence and sign commitment.

## Retrograde Inference

The software side of scrying runs **retrograde inference**:

- begin with observable anomalies, tensions, residues, and asymmetries
- run T backwards heuristically
- identify the **minimum-surprise causality chain**
- present a refined local ξ signal to the user

This is the operational form of time reversal in the framework. T-reversal is not physically rewinding the universe. It is computationally inferring the most coherent path backward from what is observed now.

The software's job is not to generate certainty. It is to reduce the search space of plausible causal chains so the user's sign commitment is less noisy.

- **Residue blindness:** failing to update when the world's response shows the last sign commitment decreased coherence

All five failure modes collapse the loop back into ordinary irreducibility.
